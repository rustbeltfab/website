How to Add Your Own Photos

1. Put your photo files (.jpg or .png) in the same folder as this HTML file.
2. In the gallery grid below, find any .gallery-item-bg div.
3. Add a style attribute like this:
style="background-image: url('your-photo-name.jpg'); background-size: cover; background-position: center;"
4. Delete the placeholder icon and text inside that div.
5. Update the overlay title and tag to describe the job.

Tip: Use wide class for landscape/horizontal photos, tall for portrait/vertical ones.
